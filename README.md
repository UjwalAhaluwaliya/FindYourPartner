A Spring Boot-based web application where users can register, login, update their profile, and view other profiles they are interested in.

🚀 Features
✅ User registration
✅ User login authentication
✅ View all registered profiles
✅ Express interest in other profiles
✅ Update own profile details

🛠 Tech Stack
Java Spring Boot
Spring MVC
Spring Data JPA
Hibernate
MySQL (or your database)
JSP (Java Server Pages)
Tomcat server

💻 How to Run
Clone the repository:
git clone https://github.com/UjwalAhaluwaliya/FindYourPartner.git
cd FindYourPartner

Open in IntelliJ or Eclipse as a Maven project.
Configure database in src/main/resources/application.properties.
Run the application
Via IDE → Run FindYourPartnerApplication.java

Or via terminal:
mvn spring-boot:run

Access in browser
http://localhost:8080/

✨ Screens (JSP Views)
Login.jsp → User login page
Register.jsp → New user registration
home.jsp → Home dashboard listing profiles
interestedIn.jsp → View profiles you showed interest in
myProfile.jsp → View your own profile
updateProfile.jsp → Update your details
